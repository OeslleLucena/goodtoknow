# Tic Tac Toe | Minmax project

This Tic Tac Toe game is a personnal project in order implement a simple minmax algorithm.

## Installation

### Requirements

* Python 3
* numpy 1.14.0
* pygame 1.9.3

```bash
$ pip install -r requirements.txt
```

## Usage

```bash
$ python3 tictactoe.py
```
